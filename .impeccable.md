# BioHackMama — Design Context

## Design Context

### Users

Polki w wieku 25–45 lat, które traktują swoje zdrowie poważnie i chcą rozumieć mechanizmy swojego ciała — nie tylko "jeść zdrowo", ale optymalizować hormony, sen, energię na podstawie nauki. Korzystają z telefonu i laptopa, często wieczorami po pracy. Poszukują wiedzy, której nie daje im lekarz pierwszego kontaktu. Chcą czuć się zainspirowane i zmotywowane — "chcę to zrobić, zaczynam dziś" — nie przytłoczone lub zagubione.

### Brand Personality

Naturalna, holistyczna, ciepła.

Marka to naukowiec-przyjaciółka, która mieszka przy morzu i pisze notatki naturalnym tuszem. Mówi o hormonach i biohackingu tak, jak opowiadałaby o gotowaniu — z pasją, precyzją i troską. Nie jest zimna ani kliniczna. Nie jest też influencerką z różowymi balonami. Jest autentyczna, kompetentna i serdeczna.

### Aesthetic Direction

**Motyw**: Jasny (light mode only). Strona jest przeglądana w ciągu dnia, w spokojnych momentach — poranek z kawą, popołudnie na kanapie.

**Paleta**: Coastal Wellness — już ustalona i spójna. Ciepły piasek (#EFEAE4), stłumiony złoty (#A68A69), oceański niebieski (#7A90A8), głęboki łupek (#48596B). Nie dodawać nowych kolorów poza tym systemem.

**Typografia**: Cormorant Garamond dla nagłówków (elegancja, lekkość), Inter dla treści (czytelność, zaufanie), Playfair Display dla akcentów. Fonty z listy zakazanych impeccable — Playfair i Cormorant się na niej znajdują — ale są już głęboko zakorzenione w marce i nie zastępujemy ich. Zachowujemy, doprecyzowujemy ich użycie.

**Inspiracje/feel**: Jak premium wellness editorial — Kinfolk Magazine, Patagonia campaigns, wysokobudżetowe strony naturalnych suplementów, butikowych retreatów zdrowotnych. Ciepło, przestrzeń, substancja.

**Anti-referencje**:
- Korporacyjna strona SaaS (dark mode, neon, dashboard UI, tech startup)
- Apteka / klinika (sterylna biel, zimny błękit, medyczny vibe)
- MLM wellness influencer (pastelowe różowe, serduszka, "kochanie")
- Typowy blog zdrowotny (zielone smoothie, stock photos kobiet w lesie)

### Design Principles

1. **Ciepło zamiast sterylności** — Unikaj zimnych bieli, ostrych kontrastów, klinicznych układów. Każda powierzchnia powinna oddychać ciepłem piasku i złota.

2. **Zaufanie przez substancję** — Treść i nauka prowadzą, nie dekoracja. Elementy wizualne służą komunikacji, nie efektowi "wow". Typografia > ilustracje.

3. **Kobiece bez infantylizacji** — Eleganckie, poważne, naturalne. Nie różowe, nie słodkie, nie dziewczęce. Femininość przez proporcje i subtelność, nie przez motywy.

4. **Inspiracja do działania** — Każda sekcja powinna naturalnie prowadzić do kolejnej. Hierarchia wizualna nakierowuje uwagę: "dowiedz się więcej → zacznij kurs → zmień coś dziś".

5. **Bez SaaS-estetyki** — Żadnych ciemnych motywów, glow-efektów, tech-stackowych layoutów, neonowych akcentów, dashboardów. To jest wellness, nie startup.

### Technical Constraints

- Framework: Next.js 14 App Router + Tailwind CSS
- Animacje: Framer Motion (już zdefiniowane easing curves — `--ease-out-strong`, `--ease-spring`)
- Nie dodawać nowych fontów — system 3 rodzin jest zamknięty
- Trzymać się tokenów z `coastal-*` namespace
- Light mode only — brak dark mode
- Obrazy przez Sanity CDN + Next.js Image
- WCAG AA minimum dla kontrastu tekstu
